# Web Art Blog

This is a personal web art blog built as part of the Udacity Frontend Developer Nanodegree program. The project includes a dynamic layout featuring a variety of art posts and interactive elements, with responsive design for various screen sizes.
This visually engaging web art blog showcases a collection of articles and content related to prose, lyric, quotes, digital art and web design. The site features a dynamic layout, smooth animations, and includes a small digital clock feature as an added touch.

## Features:

- **Responsive Layout**: The blog adapts to different screen sizes using CSS Grid and Flexbox, ensuring a seamless experience across devices.
- **Search Functionality**: Users can filter articles based on keywords using a search form,
- **Interactive Cards**: Articles are displayed within interactive cards with hover effects.
- **Sticky Sidebar with Links**: The sidebar with social media links stays fixed while scrolling.
- **Article Search**: Easily filter and find articles based on search queries,
- **Digital Clock**: A small digital clock feature that shows the current time in 12-hour format (AM/PM).

## Technologies Used:

- **HTML**: For the structure of the web page and content.
- **CSS**: To create responsive layouts, animations, and styling for the page elements.
- **JavaScript**: Used for dynamic content like the search functionality and the digital clock.
- **Responsive Design**: Using media queries to ensure the blog is mobile-friendly.

## Digital Clock Feature:

- Displays the current time in a 12-hour format with AM/PM.
- Updates every second.
- Integrated into the blog for a modern, real-time feel.

## How It Works:

1. **HTML**: The page is structured with sections for the blog articles, sidebar, and header.
2. **CSS**: The layout is responsive, with grid and flexbox implementations to ensure content is displayed appropriately across various devices.
3. **JavaScript**: Powers the dynamic search functionality for articles and updates the digital clock every second.

## Getting Started:

1. Clone or download the repository.
2. Open the `index.html` file in your browser to see the blog in action.
3. Use the search feature to find articles or simply browse the collection.
4. The digital clock will display in the corner of the page.
